# Frontend — Nettoyage Plus

Pré-requis: Node.js (16+) et npm ou pnpm.

Install & run:

```bash
cd frontend
npm install
npm run dev
```

Build / preview:

```bash
npm run build
npm run preview
```

Notes:
- La configuration utilise Vite + React + TypeScript + TailwindCSS.
- Après `npm install`, Tailwind sera prêt (config incluse). Pour connecter le frontend au backend, ajoute l'URL du backend dans un fichier `.env` et crée un service API dans `src/api`.
