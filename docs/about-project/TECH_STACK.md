# Tech Stack - Nettoyage Plus

Last Updated: December 27, 2025

## Frontend - Web (ERP Interface)

Framework: React.js
Styling: TailwindCSS
Language: JavaScript/TypeScript

## Backend - API

Framework: NestJS (Node.js)
Language: TypeScript
Architecture: RESTful API

## Database

Database: PostgreSQL
Hosting: Supabase (cloud)

## Mobile App

Framework: React Native
Platform: Expo

Primary: Firebase

Purpose: Images, documents, attachments

Service: Firebase Cloud Messaging
Purpose: Push notifications to mobile
